STAGE 1: HTML FOUNDATIONS (Day 1–2)
🎯 What HTML Really Is

HTML = HyperText Markup Language

It is NOT programming.
It is a structure language used to tell the browser:

“This is a heading, this is a paragraph, this is an image…”

🧠 How the Web Works (Simplified)

You write HTML

Browser reads it

Browser renders UI

🧱 Basic Structure of Every HTML Page
<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>

  <body>
    <h1>Hello World</h1>
    <p>This is my first page</p>
  </body>
</html>
🔍 Breakdown:

<!DOCTYPE html> → tells browser it's HTML5

<html> → root of page

<head> → settings (not visible)

<body> → visible content

What is an HTML Element?

An HTML element is everything from the opening tag to the closing tag:

<p>This is a paragraph</p>

👉 This whole thing is ONE element.

🧱 Structure of an Element
<tagname attribute="value">Content</tagname>

Example:

<a href="https://example.com">Visit</a>

<a> → opening tag

href → attribute

Visit → content

</a> → closing tag

🔍 TYPES OF HTML ELEMENTS

There are 3 main types you must understand:

1️⃣ Normal Elements (Container Elements)

They have:

Opening tag

Content

Closing tag

Examples:

<div>Hello</div>
<p>Text</p>
<h1>Title</h1>
2️⃣ Empty Elements (Self-Closing Elements)

These DO NOT have closing tags
They also DO NOT contain content

⚡ Common Empty Elements
<br>      <!-- line break -->
<hr>      <!-- horizontal line -->
<img>     <!-- image -->
<input>   <!-- form input -->
<meta>    <!-- metadata -->
<link>    <!-- external resources -->

Example:

<img src="image.jpg" alt="photo">
<br>
<hr>
❗ Important Rule:

Empty elements don’t wrap anything

❌ Wrong:

<br>Text</br>

✅ Correct:

<br>
3️⃣ Nested Elements

Elements inside other elements:

<div>
  <p>Hello <b>World</b></p>
</div>

👉 This is how real layouts are built.

📦 BLOCK vs INLINE ELEMENTS (VERY IMPORTANT)

This is one of the most important concepts in HTML.

🟥 BLOCK ELEMENTS
🔥 Behavior:

Take full width

Start on a new line

Stack vertically

✅ Examples of Block Elements:
<div></div>
<p></p>
<h1></h1> - <h6></h6>
<section></section>
<header></header>
<footer></footer>
<ul></ul>
<li></li>
🧠 Example:
<p>First</p>
<p>Second</p>

👉 Output:

First
Second
🟦 INLINE ELEMENTS
🔥 Behavior:

Take only needed width

Stay on the same line

Do NOT break line

✅ Examples of Inline Elements:
<span></span>
<a></a>
<img>
<b></b>
<i></i>
<strong></strong>
<em></em>
🧠 Example:
<span>Hello</span>
<span>World</span>

👉 Output:

Hello World
⚖️ BLOCK vs INLINE (Clear Comparison)
Feature	Block Element	Inline Element
Width	Full width	Content width
Line Break	Yes	No
Layout Use	Structure	Text styling
Examples	div, p, h1	span, a, b

⚠️ IMPORTANT RULES
❌ Inline inside Block → ✅ OK
<p>Hello <span>World</span></p>
❌ Block inside Inline → 🚫 BAD PRACTICE
<span><div>Wrong</div></span>
